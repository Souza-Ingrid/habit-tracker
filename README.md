# 🧠 Habit Tracker - Python

Projeto simples desenvolvido em **Python** com o objetivo de **praticar lógica de programação e aprender conceitos básicos da linguagem**.

O sistema funciona no **terminal** e permite que o usuário gerencie hábitos do dia a dia, podendo adicionar novos hábitos, visualizar os cadastrados e marcar hábitos como concluídos.

Este projeto foi criado como parte do processo de aprendizado em programação, focando principalmente no desenvolvimento da **lógica de programação** e no uso de estruturas fundamentais do Python.

---

# 🚀 Funcionalidades

O programa possui um menu interativo no terminal com as seguintes opções:

1️⃣ **Adicionar hábito**
Permite cadastrar um novo hábito na lista.

2️⃣ **Ver hábitos**
Mostra todos os hábitos cadastrados pelo usuário.

3️⃣ **Marcar hábito como concluído**
Permite selecionar um hábito da lista e marcá-lo como concluído com um ✅.

4️⃣ **Sair do programa**
Encerra a execução do sistema.

# ▶️ Como executar o projeto

1. Clone este repositório:

```
git clone https://github.com/Souza-Ingrid/habit-tracker
```

2. Acesse a pasta do projeto:

```
cd habit-tracker
```

3. Execute o programa:

```
python main.py
```
